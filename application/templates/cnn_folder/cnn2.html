BASICS OF CONVOLUTION NEURAL NETWORK



BIOLOGICAL NEURON ARCHITECTURE



  1. Dendrite: These will receive incoming signals from other neurons
  2. Soma: This is cell body which processes the input signals and decides to fire an output signal
  3. Axon: This will get the processed signal to different cells
  4. Synapse: This will connect axon of one nueron to the other neuron dendrites

MATHEMATICAL NEURON ARCHITECTURE






  1. Inputs: This will be input for the nueron from other neuron
  2. Weights: This is the weight associated with the input, contributing to the output
  3. Bias: This is value which is added neuron function
  4. Neuron Function: This is function which performs the operation on the Inputs, Weights and bias
  5. Activation function: This is the nonlinear function which operates on the output and provides the result
