MACHINE TRANSLATION

This model is based on Seq2seq model for Spanish to English translation.Attention methods focuses on the important word for the given sentences. Following are the steps take once the dataset is downloaded.
  1. Add start and end tokens to each sentence
  2. Clean the sentences by removing special characters
  3. Create the word index and reverse word index (dictonary mapping from word->id and id->word)
  4. Pad each sentence with max length
