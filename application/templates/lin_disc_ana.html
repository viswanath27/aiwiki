BASICS

This section explains the Linear Discriminant Analysis algorithm of the machine learning domain, its uses and its examples.

LDA is like PCA, but it focuses on maximizing the separability among known categories. This thought process should understood so deeply such that we should be able to figure out the same meaning from the mathematics to understand what brings that difference.

We can start with super simple example of reducing the 2D graph to 1D graph. one bad way to do it to ignore the Gene Y and project all th 2D to X axis.

LDA uses the information from both the axis to create a new axis. This other axis is used to project the information on to it, in a way to maximize the seperation of the two categories.In summary LDA creates a new axis and projects the data on to it such a way that it maximizes the difference of these two categories. Lets look at how LDA creates new axis.

[Criteria 1] The new axis is created according to new criteria that are considered simultaneously. The criteria is once the data is projected on to a new axis, we want to Maximize the distance between two means.

[Criteria 2] Minimize the variation (which LDA calls "Scatter" and is represented by s2) within each category.

If we have to consider these two criteria simultaneously, we have the ratio of these both criteria as shown below.

Suppose for example we only maximize the distance between means for some data sets there will be still a lot of overlap. This isn't a great seperation. So if we optimize both criteria at the same time, we get a good seperation. What if we have more than 2 Dimensions, the process still remains the same. What if we have three categories to come up with. In this case we first find the center point of data, than we measure the distance between center point of each category and the main center point.


USECASES

Little bit about the problem statement of where Linear Discriminant Analysis is useful.

Suppose there is this new cancer drug

  1. It works great for some people.
  2. But for some people it may turn out to be worse.

How do we decide who to give the drug to ? We want to give it to people whom this will help but now harm. May be gene expression can help us decide here.If we plot the gene expression on the number line and see, there can be segregation of dots which are all one side (people for which drug works) rest of dots allligned at other side (people for which drug does not work for). This is basically low transcription of gene X and druf is not working for the people who have high transcription of gene X. There can be an overlap in middle which is not a clear cutoff of who to give the drug to. In summary Gene X does an okay job to tell us who should be drug given vs whom it should not be given. Can we improve this situation, what if we can use more than one Gene to make a decision. There could be Gene X and Gene Y plotted in 2D space where it shows who gets the drug and who does not get. Here there can be a line in 2D space where below which drug works and the other side drug does not work. This is more clear and better than just using one Gene. How about using 3 Genes. Here we can have a plane to seperate the boundary between where the drug works and where it will not work. What if we need 4 or more Genes to seperate two categories. We cannot draw such dimension graph which is more than 3D. This is similar to PCA problem. PCA reduces dimensions by focusing on the genes with the most variation. This can be useful to plotting data with a lot of dimensions onto a simple XY plot. However in we are not interested in the genes with most variation. But we are interested in maximizing the separability between the two groups so that we can make a best decision.

If we compare LDA and PCA, LDA gives clear demarkation but PCA does not give the clear seperation. PCA does not seperate but will know the genes which has more variation.

Similarities between PCA and LDA

  1. PCA1 accounts for the most variation in the data. PCA2 does the second best job.
  2. LDA1 accounts for the most variation between the categories. LDA2 does the second best job.
